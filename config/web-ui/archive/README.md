# Archive - Obsolete Web UI Files

This folder contains old/obsolete versions of the web UI that have been replaced by the unified interface.

## Files

- **config-viewer.html** - Old table-based config viewer (replaced by config-unified.html)
- **feed-status.html** - Old standalone feed status monitor (integrated into config-unified.html Dashboard)
- **index.html** - Old config editor (replaced by config-unified.html)
- **design-mockups.html** - Design exploration mockups
- **sidebar-mockup.html** - Sidebar design mockup (implemented in config-unified.html)
- **config-unified-backup.html** - Backup of config-unified.html before adding suggestions

## Current Active Files

Located in `/config/web-ui/`:
- **config-unified.html** - Unified feed manager with sidebar navigation, dashboard, and feed management
- **email-archive.html** - Email archive browser with sidebar navigation
- **config-editor.js** - JavaScript utilities for config editing

## Why Archived?

These files were replaced in October 2025 when the web UI was redesigned with:
- Sidebar-based navigation
- Integrated dashboard view
- Feed suggestions functionality
- Consistent teal/green color scheme
- Better mobile responsiveness

Keep these files for reference but do not link to them from active pages.
